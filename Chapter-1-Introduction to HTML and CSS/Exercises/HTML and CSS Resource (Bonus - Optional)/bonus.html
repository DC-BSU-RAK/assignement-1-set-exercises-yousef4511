What is HTML?

HTML, or Hypertext Markup Language is a Markup language that is essentially the building blocks of the web. It's syntax was based off the Standard Generalized Markup Lanuage when it was invented in 1993 by Tim Berners-Lee. You can make your own website with HTML, and you often include it with Cascading Style Sheets, and Javascript. All elements are often enclosed between less than and greater than tags. Example: <element>element</element>

Headers

Headers are important for websites as they can let the user know what part of your website they are on. There are 6 types of headers in HTML. the higher the number, the smaller the header is (Tip: sizes on h1 and h2 may vary depending on your browser).

Header Examples:

  1. <h1>Header 1</h1>

  2. <h2>Header 2</h2>

  3. <h3>Header 3</h3>

  4. <h4>Header 4</h4>

  5. <h5>Header 5</h5>
  6. <h6>Header 6</h6>

PROTIP: It is recommended to have only one h1 per page in HTML.

Text

With the p tag, you can add text to your website. You can also format it, like making it bold, italic, underlining it, etc. You'll learn that in the next section. Example below:

<p> This is text in a paragraph tag </p>

Text Formatting

With tags like i, b, or strong, you can also make your text look fancy. Examples:

<i> Italic Text </i>

<b> Bold text </b>

<strong> Another way of making text bold. </strong>

<ins> Inserted text </ins>

<del> Deleted text </del>

<sup> Supscripted text </sup>

<sub> Subscripted text </sub>

<small> Small text </small>

Links/Hyperlinks/Anchors

You can add anchors to your website to link to another website of your choice. Example:

<a href="google.com"> Google.com </a>

Or if you want it to open in a new tab:

<a href="google.com" target="_blank"> Google.com </a>

Buttons

Buttons help give your website some interactivity. Use attribute onclick to make the button perform an action upon clicking it. You can also add text inside the button. Example

<button onclick="alert("Hello World!")"> </button>

Comments

To make a comment, simply type <!-- put text here -->. Comments are ignored in HTML and are often used to help label parts of HTML markup.

Tables

Tables can help you organize stuff, like in Spreadsheet. Simply type <table border="1"> <tr> <td>One</td> </tr> </table>. Add as many td's as you want. Add another tr to start a new table row.

Example:

Semantic Tags

Semantic tags are tags that make your HTML accessible to screen readers and easier to parse. While they are not required when making your website, they are recommended. Some semantic tags are:

  1. <main></main>, for main elements in your HTML.
  2. <nav></nav>, for unordered or ordered lists.
  3. <header></header>, for headers.
  4. <article></article>, for articles.
  5. <section></section>, for seperating certain sections of your website.
  6. <aside></aside>, for stuff that is not part of the main content in your web page.
  7. <footer></footer>, for credits (i.e authors, sources).
